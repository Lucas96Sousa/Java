## Generics

* Generics permitem que <strong> classes, interfaces e métodos</strong> possam ser parametrizados por tipo. Seus benefícios são:
    - Reuso
    - Type safety
    - Performance

* Uso comum: coleções

```java
   List<String> list = new ArrayList<>();
   list.add("Maria");
   String name = list.get(0); 
```
