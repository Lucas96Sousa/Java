# Referencia sobre interface

<p>Interface é um tipo que define um conjuto de operações que uma classe deve implementar.</p>

<p> A interface estabelece um <b>contrato</b> que a classe deve cumprir.<p>

```java
interface Shape {
    double area();
    double perimeter();
}
```

<strong> Pra quê interfaces ?</strong>

* Para criar sistemas com baixo acoplamento e flexíveis.
