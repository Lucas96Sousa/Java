# Enunciado da questão

```sh
Uma empresa possui funcionários próprios e terceirizados.
Para cada funcionário, deseja-se registrar nome, horas trabalhadads e valor por hora.
Funcionários terceirzados possuem ainda uma despesa adicional.

O pagamento dos funcionários corresponde ao valor da hora multiplicado pelas horas trabalhadas, 
sendo que os funcinários terceirizados ainda recebem um bonus correspondetente a 110% de sua despesa adicional.

Fazer um programa para ler os dados de N funcionários (N fornecido pelo usuário) e armazená-los em uma lista.
Depois de ler todos os dados, mostrar nomee e pagamento de cada funcionário na mesma ordem em que foram digitados.
```
